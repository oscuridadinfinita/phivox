# PhiVox: Música Dimensional Cuántica

**PhiVox** es una aplicación de música dimensional cuántica que integra la proporción áurea (Phi), la calibración por voz, los ritmos binaurales y la visualización fractal. Desarrollada en Python sobre Debian 12, utiliza la API de DashScope (Alibaba Cloud) para generar paisajes sónicos personalizados.

Este proyecto es una síntesis inédita de tres legados:
- **Cultura Chorrera (1300-300 a.C.)**: Maestros de la acústica ancestral que esculpieron el sonido de la biodiversidad en cerámica.
- **Nikola Tesla**: Visionario de la energía, la frecuencia y la vibración como claves del universo.

PhiVox es la herencia viva de estos tres gigantes, unificada en código abierto.

## 🛠️ Estructura del Proyecto

| Archivo | Función |
|:---|:---|
| `config.py` | Fuente única de verdad: constantes, rutas, modelos y parámetros. |
| `acoustic_engine.py` | Motor de síntesis de audio: secuencias áureas, beats binaurales, síntesis OM. |
| `quantum_analyzer.py` | Analizador de voz: extrae la frecuencia fundamental y el perfil vibracional. |
| `fractal_visualizer.py` | Visualizador de espiral áurea en tiempo real. |
| `app.py` | Orquestador Flask + Socket.IO: endpoints REST y streaming WebSocket. |
